¿Qué es un Buen Prompt?

20-25 min

Un prompt es...

La instrucción que le das a la IA para que entienda exactamente lo que quieres crear

Prompt malo

"Haz una app"

  • Muy vago
  • Sin detalles
  • Resultado impredecible

Prompt bueno

"Crea una app para adolescentes que ayude a rastrear hábitos de estudio diarios con recordatorios y estadísticas visuales"

  • Específico
  • Define el usuario
  • Explica la función
